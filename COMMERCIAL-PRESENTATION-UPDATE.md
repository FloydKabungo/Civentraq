# Civentraq commercial presentation update

This build adds the requested report-governance, presentation-editor, sharing, and quality-control features.

## Included

- Separate operational project-health and insight-review scores
- Cost, schedule, safety, and risk health dimensions
- Reusable locked master slide templates
- Snap-to-grid, guides, multi-select, grouping, z-order, locking, copy/paste, zoom, and keyboard shortcuts
- Draft, client-review, approved, and final report versions with restore, duplicate, and compare
- Slide and element comments, change requests, approvals, and workflow history
- Dataset source, refresh date, reporting period, filters, and source-status labels on charts and KPIs
- Improved PowerPoint/PDF rendering with notes and widescreen, A4 landscape, and A4 portrait layouts
- Encrypted read-only client links with password, expiry, download permission, views, and comments
- Five-step first-use onboarding wizard
- Fixed Civentraq product identity and fictional demonstration project/client names
- Pre-export checks for empty slides, chart titles, source metadata, stale data, low-resolution photos, text overflow, colour inconsistency, and unresolved comments

## Static prototype limitation

The site is still a browser-only prototype. Versions, comments, views, and client feedback are stored locally. Cross-device tracking, central access control, shared comments, and secure hosted links require a backend in a later phase.
