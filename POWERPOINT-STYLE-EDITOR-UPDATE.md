# PowerPoint-style editor update

The Presentation Studio has been reorganised into a familiar PowerPoint-style workspace.

## Main improvements

- Home, Insert, Design, Review, and View ribbon tabs
- Larger working canvas with the properties pane collapsed by default
- Slides-only thumbnail panel to reduce clutter
- New Slide and Layout controls with reusable templates
- Direct font, size, bold, italic, underline, colour, alignment, bullets, and spacing controls
- Shape fill, outline, and corner controls
- Insert gallery for headings, text, AI summaries, KPIs, charts, tables, photographs, comparisons, progress bars, and callouts
- Theme previews, slide-size selection, slide background, and master slide access
- Review tools for versions, comments, approvals, presentation checks, and data sources
- View controls for slide thumbnails, format pane, gridlines, snapping, fit-to-window, and zoom
- Collapsible ribbon, collapsible notes, and a PowerPoint-style bottom status bar
- Improved split-screen and laptop responsiveness
- Zoom range increased to 30%–200%
- A4 portrait canvas fitting corrected

All existing Civentraq report data, exports, comments, versions, master templates, sharing, checks, and governance features remain available.
