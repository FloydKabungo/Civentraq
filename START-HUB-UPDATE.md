# Civentraq Start Hub

This update changes the first screen of the application.

## New opening flow

1. The site opens on the Civentraq Start Hub.
2. Users can create a project, import data, use a template, or open a recent project.
3. Selecting **New project** opens a four-step setup workspace rather than the active dashboard.
4. The project dashboard only opens after the project setup is confirmed.

## Main interface additions

- Home sidebar and global search
- Unique industrial project-intelligence hero
- Recent project cards
- Popular reporting templates
- Full projects library
- Project switching
- Guided project-creation wizard
- Data-start options: sample, blank, or import after creation
- Executive, commercial, delivery, and refinery template choices
- Dashboard-to-home navigation through the Civentraq logo
