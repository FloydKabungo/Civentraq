## Phase 2 boundary

These items intentionally remain outside the static Phase 1 prototype:

- Secure user authentication and multi-factor authentication
- Central cloud database and file storage
- Server-side company and project permissions
- Real-time shared comments and view analytics across devices
- Email, Teams, and SMS notifications
- Live SAP, Primavera, SharePoint, and Microsoft 365 integrations
- Enterprise audit logs, backup, disaster recovery, and security certification

Phase 1 is ready to be used as the front-end product prototype and as the design foundation for the Phase 2 backend build.
